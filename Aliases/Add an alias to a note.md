To add an alias for a note, add `aliases` property in the note [Properties](https://help.obsidian.md/Editing+and+formatting/Properties). Aliases should always be formatted as a list in YAML.

```md
---
aliases:
  - Doggo
  - Woofer
  - Yapper
---

# Dog
```
